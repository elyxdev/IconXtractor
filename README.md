# iconXtractor
## ¿Qué es?
- Extractor de íconos sencillo y ligero para Windows. Basado en Python.

## Modo de uso
- Arrastra y suelta el archivo .exe objetivo sobre iconXtractor. El ícono quedará en la misma carpeta del archivo objetivo.
